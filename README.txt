***** Project Description: ******
Create an auction website like EBAY.com that allows users to list items and handle bidding. This auction website also has a login in system for both end-user, admins, and customer representatives. Admins are allowed to create reps, reps are allowed to delete/edit end-users bid items/ accounts, and users are allowed to list/place bids.

#Technologies Used
Using HTML for the user interface, MySQL for the database server, and Java, Javascript, and JDBC for connectivity between the user interface and database server. 

* SQL (Tomcat)
* JDBC
* Java / Javascript
* HTML, CSS
